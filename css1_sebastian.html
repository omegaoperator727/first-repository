Principios Básicos de CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño usado para cambiar la apariencia de documentos web. Con CSS, puedes controlar el color, el tamaño, el espacio, y mucho más.

¿Qué es CSS?

CSS es la capa de presentación de las páginas web. Junto con HTML y JavaScript, forma una de las tres tecnologías clave para la creación de sitios web. Algunos de sus usos más comunes incluyen:

  • Colores: Cambiar el color de texto, fondos o bordes.
  • Fuentes: Cambiar el tipo, tamaño y estilo de la fuente.
  • Espaciado: Ajustar márgenes, rellenos y el espacio entre elementos.
  • Diseño: Crear diseños responsivos y alineaciones complejas.

Ejemplo de una regla CSS

Una regla CSS típica sigue la siguiente estructura:

        selector {
            propiedad: valor;
        }

Por ejemplo, la siguiente regla cambia el color de los encabezados a<h1>Esto es un encabezado </h1> a azul:

        h1 {
            color: blue;
        }

Propiedades más comunes de CSS

  • color: Define el color del texto.
  • background-color: Cambia el color de fondo.
  • font-size: Cambia el tamaño de la fuente.
  • margin: Ajusta el espacio exterior de un elemento.
  • padding: Ajusta el espacio interior de un elemento.
  • border: Añade bordes a un elemento.
